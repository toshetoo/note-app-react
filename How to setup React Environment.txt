За да можете да пишете реакт на вашата машина имате нужда от следния софтуер:

NodeJS >= 6 , може да бъде свален от тук => https://nodejs.org/en/download/
NPM  5.2.0+ => имате го by default след инсталация на NodeJS
VisualStudio Code, свалено от тук => https://code.visualstudio.com/download

След инсталиране на горните три софтуера, отворете Command Prompt и изпълнете следната команда:
npm install -g create-react-app

Това ще зареди Boilerplate app софтуера, който лесно ще ви позволява да създавате/рънвате React Applications
Създаването на нова реакт апликация става по следния начин:
- Отваряте Command Prompt
- create-react-app името-на-вашата-апликация
- npm install


Вече ще имате готова папка със нужните неща, заедно със базов проект с няколко файла. За да стартирате проекта, отваряте Command Prompt в новосъздадената директория и изплнявате 
npm run start

Ако отидете във браузъра си и опитате да достъпите адрес http://localhost:4200 би трябвало да видите вашата апликация.
